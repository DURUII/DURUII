## 项目经历

人工智能创新大赛：多模态信息融合影视台词说话人智能标注
~ 2025.08 – 2025.10
- 项目背景：在影视长视频与二创剪辑中，字幕普遍缺失“说话人 / 角色”信息，给内容理解、二次配音创作带来较高成本。项目面向真实影视剧场景，探索利用 LLM + 声纹 + 视觉 的多模态方式，实现台词级说话人自动标注。

- 研究对象：以电视剧《三体》为样本，对前 15 集进行规模化自动标注，覆盖多时间线叙事、多角色密集对话、画外音与超短语音等复杂场景。

- 核心方案：构建“字幕 → 语义推断 → 声纹 / 人脸校验”的三模态融合流水线，在开源 3D-Speaker 框架基础上进行工程化改造。

  - 语言模态（字幕 & LLM）：基于 PaddleOCR 提取影视硬字幕并生成时间戳台词序列；设计 LLM + 工作流 的字幕清洗方案，自动过滤广告、水印、乱码等噪声；结合剧情摘要与角色信息，利用大模型完成说话人切换、场景切换与角色推断；通过分块与上下文重叠策略，解决长视频字幕超出模型上下文窗口的问题。

  - 声纹与视觉模态：基于 3D-Speaker 构建主要角色声纹库，支持开放集 “other” 角色判别；构建角色级人脸嵌入库，用于弥补影视中大量 <1 秒短语音 场景的声学不足；针对跨模态分布差异，引入归一化与尺度校准策略。

  - 多模态融合与工程实现：采用 Late Fusion 融合声纹与人脸相似度，结合阈值与 top-k 差异判定，降低误匹配风险；通过 API 批量调用与结果持久化，实现 15 集电视剧的自动化标注流水线；基于 React + TypeScript 实现 RTTM 可视化工具，支持 DER 误差分析与快速调参。

服创大赛：高性能声纹识别平台，项目负责人（2023.02）
~ [<span class="iconify" data-icon="simple-icons:baidu"></span> 文档&代码](https://pan.baidu.com/s/1MJ7-E8cWqtXI47gY1kxcHA?pwd=duru), [<span class="iconify" data-icon="ri:bilibili-fill"></span> Demo](https://www.bilibili.com/video/BV1y5411v7Xi/)
- 本项目旨在开发一个高性能的声纹识别平台，利用音频信号处理技术和声纹识别技术，实现对通话用户身份的自动确认，以节省成本、提高用户满意度。
- 技术栈：深度学习框架：PyTorch；音频处理：torchaudio、numpy、scipy、SoX、Kaldi；数据集：VoxCeleb1、VoxCeleb2、aidatatang_200zh、MUSAN、RIR_NOISE
- **任务难点**：要实现声纹识别，不是简单的分类问题。因为声纹需要注册、验证和注销。中文数据集小，英文数据集大，能否做做移学习？是先在英文数据集上训练，再在中文数据集上训练吗？
- **技术亮点**：
  - 高准确率的声纹识别：在未见过的测试数据上，文本无关场景下，声纹确认准确率高达<u>**96.21%**</u>，等错误率低至<u>**3.79%**</u>，系统响应时间低至1.7秒。
  - 多种数据增强手段：通过加性噪声、混响、节奏、音量、音调等数据增强方法，扩充训练数据，提高模型的泛化能力和鲁棒性。
  - 使用大规模数据集进行预训练，并融合问题域数据进行微调，采用动态学习率策略和多种优化技术（如残差结构、SE block、ECAPA-TDNN）以提高模型性能。采用预热和阶梯下降的动态学习率调整策略，加速模型收敛，提高模型性能。
  - 平台支持多种语言、方言和复杂语音环境，具有良好的适应性和扩展性。
